POST /api/analyze

Analyze a list of reviews to get the top 5 most positive ones

Request Body:
{
  "reviews": ["Amazing place!", "Great location!", ...],
  "propertyName": "My Property"
}

POST /api/scrape-reviews

Scrape reviews from an Airbnb listing (may use mock data on serverless platforms)

Request Body:
{
  "airbnbUrl": "https://www.airbnb.com/rooms/12345678",
  "maxReviews": 10
}

POST /api/scrape-and-analyze

Scrape reviews and analyze them to get the top 5 positive reviews

Request Body:
{
  "airbnbUrl": "https://www.airbnb.com/rooms/12345678",
  "maxReviews": 20
}
